# BoundedOverlayManager

## Overview

`BoundedOverlayManager` is a React library designed to create and manage overlay components within a specified bounding container. It offers flexible positioning, dynamic show/hide behavior, customizable event handling, and an API for extended control.

## Key Features

- **Flexible Positioning**: Leverage `position` and `offset` parameters of Overlay component for precise placement of overlays.
- **Dynamic Visibility**: Configure overlays to appear and disappear in response to user interactions.
- **Customizable Event Handling**: Tailor the behavior of the overlays to match specific application requirements.
- **Extended Control via API**: Utilize the provided API for more complex and dynamic interactions.

## Installation

```bash
npm install react-bounded-overlay-manager
```

## Basic Example

```jsx
import React, { useRef } from 'react';
import BoundedOverlayManager, { Overlay, PredefinedPosition } from 'react-bounded-overlay-manager';

const BasicExample = () => {
  const boundingComponentRef = useRef(null);

  return (
    <div>
      <div ref={boundingComponentRef} style={{ width: '70vw', height: '50vh', border: '1px solid black' }}>
        Hover over this div to display overlays.
      </div>
      <BoundedOverlayManager boundingComponentRef={boundingComponentRef}>
        <Overlay position={PredefinedPosition.BOTTOM_CENTER}>
          <button>Overlay Button</button>
        </Overlay>
        <!-- more overlays here -->
      </BoundedOverlayManager>
    </div>
  );
};
```
